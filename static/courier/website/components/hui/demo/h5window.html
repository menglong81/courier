HUI 创建并打开窗口

说明
hui.open(winName, styles, isClose, extras)
功能 : 打开一个webview 并展示出来（如果窗口不存在会自动创建它）
参数 : 
1、winName ulr地址 
2、窗口样式 {}对象形式, 
3、返回时窗口处理方式（可选参数 默认false 代表只隐藏不关闭窗口，传递true代表关闭窗口）
4、extras 创建窗口是附带的数据 (对象格式)
返回 : 创建后的窗口对象
关于窗口ID : 系统使用第一个参数 winName ulr地址 作为窗口ID
关于窗口层级 : 使用hui.open打开的窗口默认 zindex 值为10
